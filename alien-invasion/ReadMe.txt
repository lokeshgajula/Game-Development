Alien Invasion.

simple ,yet interesting game. This game has its own legacy. One of the classic arcade game.
I used python pygame module to develop the game.
all the code and game resources are uploaded.


How to run the game:
you need python3 or above installed in your pc.
next install pygame using the command "pip install pygame" in your terminal.
next run the alieninvasion.py file. the game should run automatically.
